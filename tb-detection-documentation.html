TB Detection System Documentation

1. System Overview

The TB Detection System is an AI-powered application that analyzes chest X-rays to detect tuberculosis. It combines deep learning with blockchain technology for secure record-keeping.

Key Features:
  • AI-based TB detection from X-ray images
  • Web-based user interface
  • Blockchain record storage
  • 9-class TB classification

2. System Requirements

Software Requirements:

  • Python 3.x
  • TensorFlow
  • Flask
  • OpenCV
  • Web3.py
  • Ganache (Blockchain)

Hardware Requirements:

  • Minimum 8GB RAM
  • Storage space for model files (~100MB)
  • GPU (optional but recommended)

3. Installation Steps

  1. Install Python dependencies: pip install tensorflow flask opencv-python web3 numpy
  2. Install Ganache for blockchain simulation
  3. Clone project repository
  4. Set up environment variables

4. Project Structure

project/
├── app.py (Main Flask application)
├── model.py (AI model implementation)
├── templates/
│   ├── index.html (Upload interface)
│   └── result.html (Results display)
├── static/
│   └── uploads/ (Image storage)
└── blockchain/
    └── contracts/ (Smart contracts)

5. How to Use

  1. Start Ganache blockchain simulation
  2. Run Flask application: python app.py
  3. Access web interface at: http://localhost:5000
  4. Upload X-ray image
  5. View results and blockchain record

6. Features

Image Processing:

  • Automatic image resizing to 100x100 pixels
  • Image normalization
  • Support for common image formats

AI Model:

  • Convolutional Neural Network (CNN)
  • 9-class classification
  • High accuracy detection

Blockchain Integration:

  • Secure record storage
  • Transaction tracking
  • Impact counter

7. Troubleshooting

Common Issues:
  • Model Loading Error: Verify full_model.h5 exists
  • Blockchain Connection: Check if Ganache is running
  • Upload Issues: Verify upload directory permissions

8. Classification Categories

  • PTB (Pulmonary Tuberculosis)
  • Normal
  • ATB (Active Tuberculosis)
  • Tuberculosis Pleuritis
  • NATB
  • STB
  • TB
  • Right Upper Pneumonia

9. Security Considerations
